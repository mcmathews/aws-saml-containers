## Testing
- Open Firefox add-ons manager
- Debug mode
- Load temporary add-on
- Choose the `manifest.json` file in the root of the project

## Publishing
- `npx web-ext lint`
- `npx web-ext build --overwrite-dest`
- Go to https://addons.mozilla.org/en-US/developers/addon/aws-saml-containers/edit
- Upload New Version
